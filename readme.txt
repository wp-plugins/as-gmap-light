== Description ==

<strong>As Gmap Light</strong> is a simple map image generator plugin. You can add your own map with it via shortcode and widget. 



After installing Plugin you will get a widget which name is <strong>AS gMap Widget</strong> on your wordpress Admin panel. Put it any of your widget area with it's configuration. You can find more information <a href="http://wdashik.com/plugins/as-gmap-light/">here </a>
